1. Basics

Print

  • Once you've opened a python interpreter you can type any valid python code and hit enter. The canonical starting point is the "Hello World". Paste the following into your interpreter (after the >>>) 
print "Hello World"

Python syntax is designed to be very readable. We have a statement, print, followed by the argument, telling it what to print. In this case, we passed the string "Hello World", but you can pass any type of object to the print statement, and python will spit a string representation of that object to the console.

Since "Hello World" is already a string, print just spits it to the console. But lets pass some other types of objects to the print statement. Other object types include:

type python name
integer int
floating decimal point float
booleans bool


Let's print some of them!

# lets print an integer
print 2

# an alternative way to print (this looks like a more traditional function call)
print(2)

# lets print a float
print 2.1

# how about a boolean?
print True

How does python know how to print all these different types of objects?

  • Every object in python has string representation of itself. When you pass any object to the print statement, it converts it to that string, and displays it to you. This might seem simple, but its a fundamental concept. print takes any kind of object and displays it's string interpretation.

How can we know the type of an object?

  • Just like with the print statement, python has a very intuitive way of doing this. It's called the type function. functions are slightly different than statements, but for this tutorial, the only difference you need to know is that a function is called with parens (), and it must return something.

Type

# how do we know that 2.1 is really a float?
type(2.1)

# type can be called on any type!
type(True)
type("Hello World")

In the same way that every python object has a string representation, every python object must also have a type. Just as the print function returns the string interpretation, type returns the object type. It's a very useful command!

Variables

Creating a variable in python is also straightforward:

# lets assign your name to a variable
my_name = "John"

In python, the = sign is the assignment operator. It takes the value on the right, and assigns it to the variable on the left. In this case, we are assigning the string "John", to the variable my_name. This variable now references our string object. This means that any time you use my_name you are pointing to the string with your name in it. You can use the variable anywhere you could have used the string:

# we can print the object referenced by my_name
print(my_name)

# we can ask what type of object is stored in my_name
type(my_name)

Operations

Python tries to interpret what you mean, more than most other languages. We can see examples of this in arithmetic operations:

# lets store some numerical data
my_first_int = 4
my_second_int = 3

# we can do some simple arithmetic
print(my_first_int + my_second_int)

# we can even store the output in a new variable
my_sum = my_first_int + my_second_int

# what type will the new variable be?
type(my_sum)

Arithmetic and Types

  • when you tell python to divide two integers, it applies integer division. The division will be rounded down to the nearest integer
a, b = 3, 4
ratio = a / b

print(ratio)
print(type(ratio))
  • when you tell python to divide two floats, it can return a float
a = 3.0 
b = float(4)

print(a / b)
print(type(a / b))

Arithmetic and Mixed Types

  • when you tell python to divide a float and an int, it assumes you want a float, and converts the integer into a float before division
a, b = 3, 4.1

print(a + b)
print(a / b)

3. Functions

  • functions are simple to define and use in python
def add_two_things(thing1, thing2):
  return thing1 + thing2
  
print(add_two_things(1,2))
  • If you do not include a return statement, the function will return something called a None by default. This is python's way of denoting an empty, or valueless object.

4. Lists

  • python lists are a powerful and flexible tool for data manipulation
my_list = []
for i in range(1,6):
  my_list.append(i)
  
print my_list

List Properties

my_list = [1,2,3,4,5]
  
# first element
print my_list[0]
# last element
print my_list[-1]
# number of elements
print len(my_list)
# range of elements
print my_list[0:3]
# right-bounded range of elements
print my_list[:3]

Indexing Lists

5. Packages and Numpy

  • Numpy is a package that makes many mathematical operations simple in python.
import numpy as np
# np arrays behave like lists
print np.array([1,2,3])
# but they can be easily manipulated with mathematical operations
print np.array([1,2,3]) + np.array([1,2,3])
print np.array([1,2,3]) * np.array([1,2,3])
# they can be indexed just like regular lists
print a[:2]
  • Numpy makes dealing with matrices easy!
# the numpy function produces a N x M matrix of 0's
l = np.zeros((3,4))
print l
# indexing is done like this
print l[1,1]
# you can also give ranges to partition
print l[1:3,1]
# the "eye" function produces N x N dimensional identity matrices
M = np.eye(2)
print M
# Numpy has a large array of matrix related functions, such as the dot product
np.dot(M, M)
